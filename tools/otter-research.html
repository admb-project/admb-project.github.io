AD Model Builder This software package should be of interest to anyone who wants to create nonlinear statistical models to analyze data. It is very fast!. If you find Statistical packages such as S-Plus or Matlab are too slow you owe it to yourself to check this out! Free download of evaluation version for MINGW32 C++ or Visual C++ version 5.0 and 6.0 or Borland C++ version 5.5. Runs on Redhat Linux 5.2 and 6.x too.

ADMB-RE Nonlinear random effects module. Brings AD Model Builder's rapid development technology to the creation of nonlinear random effects models.

MULTIFAN This software package has been used to estimate growth and mortality for fishes and other species using length frequency data. A list of references to papers in which MULTIFAN was used is available.

We are currently developing an extension of MULTIFAN, called MULTIFAN-CL, which uses catch-at-length data from exploited fisheries. To date MULTIFAN-CL has been used to analyze data from two species of tuna, albacore (Thunnus alalunga) and yellowfin (Thunnus albacores) as well as Pacific cod (Gadus macrocephalus) MULTIFAN-CL. A report of the application to albacore is available as a zipped PDF file, ALBACORE.ZIP.
